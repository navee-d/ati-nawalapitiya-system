## üìä Common Tasks

### Add Students in Bulk
```
1. Create Excel file with student data
2. Import/Export ‚Üí Select "Students"
3. Upload file
4. Review results
```

### View Student List
```
1. Click "Students" in menu
2. Browse the list
3. Click "Edit" to modify
4. Use filters to search
```

### Record Exam Results
```
Method 1 - Single Entry:
1. Exam ‚Üí Add Result
2. Select student and course
3. Enter marks
4. Save

Method 2 - Bulk Import:
1. Prepare Excel with StudentID, CourseCode, Marks
2. Exam ‚Üí Upload Excel/PDF
3. Import results
```

### Export Reports
```
1. Import/Export page
2. Select data type
3. Click "Export to Excel" or "Export to PDF"
4. File downloads automatically
```

## ‚úÖ Best Practices Checklist

**Before Importing:**
- [ ] Create Excel file with correct column names
- [ ] Check for duplicate IDs
- [ ] Verify all required fields are filled
- [ ] Test with 5-10 records first

**After Importing:**
- [ ] Review the success/failure summary
- [ ] Check a few records manually
- [ ] Fix any errors and re-import
- [ ] Notify new users of their credentials

**Regular Maintenance:**
- [ ] Export data weekly for backup
- [ ] Update outdated information
- [ ] Remove duplicate entries
- [ ] Archive old records

## üÜò Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Import fails | Check column names match template |
| "Department not found" | Create department first, then import |
| Can't login | Verify email/password, check caps lock |
| File too large | Split into smaller files (max 10MB) |
| Wrong data imported | Export current data, fix in Excel, re-import |
